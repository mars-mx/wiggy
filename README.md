# wiggy

A highly opinionated Ralph Wiggum loop AI software development CLI.

## Installation

```bash
# Using uv
uv pip install wiggy

# Using pip
pip install wiggy
```

For development:

```bash
uv venv
uv pip install -e ".[dev]"
```

## Usage

```bash
wiggy --help
wiggy --version
wiggy run
```
